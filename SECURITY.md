# Security Policy

## Supported Node.js versions

This project is focused on modern Node.js versions and does not support legacy versions.

| Version | Supported          |
|---------|--------------------|
| < 18.17 | :x:                |
| 18.17.0 | :white_check_mark: |
| 19      | :white_check_mark: |
| 20      | :white_check_mark: |
| 21      | :white_check_mark: |

## Please, do not report any issue on unsupported Node.js versions
